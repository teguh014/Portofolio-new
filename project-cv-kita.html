Tentang Proyek

CV-KITA adalah platform pembuatan Curriculum Vitae (CV) yang dinamis dan interaktif. Aplikasi ini dirancang untuk membantu para pencari kerja (job seekers) menyusun resume profesional mereka secara cepat dengan berbagai pilihan template premium (Classic, Modern, Minimalist).

Fitur andalan dari sistem ini adalah live-preview, di mana pengguna dapat melihat perubahan desain dan konten CV mereka secara real-time. Hasil akhir dari CV dapat diunduh dalam format PDF berkualitas tinggi yang siap dikirimkan ke perusahaan.

Fitur Utama

  • Dynamic Form Builder untuk Data Personal & Profesional
  • Real-time CV Preview Engine
  • Multiple Design Templates (Modern, Classic, Minimal)
  • Fitur Upload & Crop Foto Profil
  • High-Quality PDF Export System

Teknologi yang Digunakan

HTML5 & CSS3 JavaScript ES6 React.js JSPDF Tailwind CSS
