Tentang Proyek

Gor Blotan System adalah aplikasi manajemen penyewaan lapangan badminton berbasis web yang dirancang untuk mendigitalisasi proses reservasi, penjadwalan, dan pengelolaan operasional GOR secara efisien. Sistem ini mengatasi permasalahan penjadwalan manual yang rentan terhadap bentrok jadwal, kesalahan pencatatan, dan proses administrasi yang memakan waktu.

Sistem menyediakan dashboard pelanggan yang memungkinkan pengguna melihat ketersediaan lapangan secara real-time, melakukan pemesanan, memilih jadwal, serta melakukan pembayaran secara online. Integrasi dengan Xendit Payment Gateway memungkinkan proses pembayaran dilakukan secara otomatis menggunakan berbagai metode pembayaran. Setelah transaksi berhasil dikonfirmasi oleh Xendit, status pemesanan akan langsung diperbarui menjadi berhasil (paid) tanpa memerlukan validasi manual dari admin, sehingga proses reservasi menjadi lebih cepat, akurat, dan efisien.

Di sisi pengelola, tersedia dashboard administrator untuk mengelola data lapangan, jadwal operasional, data pelanggan, riwayat transaksi, serta memantau pendapatan melalui laporan yang terpusat. Dengan otomatisasi pembayaran dan penjadwalan, Gor Blotan System mampu mengurangi beban administrasi, meminimalkan kesalahan pencatatan, serta meningkatkan kualitas layanan kepada pelanggan.

Fitur Utama

  • Reservasi lapangan secara online
  • Sistem Booking Lapangan secara Real-time
  • Dashboard Admin untuk Manajemen Jadwal & Pendapatan
  • ntegrasi Xendit Payment Gateway untuk pembayaran otomatis
  • Konfirmasi pembayaran otomatis tanpa validasi manual admin
  • Desain Responsif (Mobile-first approach)
